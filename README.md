Ce TP avait pour objectif de **convertir une maquette HTML Bootstrap en formulaire Symfony** et de créer une page produit permettant d’**ajouter un article au panier**.

**Fichiers inclus :**  
- `src/Controller/ProductController.php` — contrôleur gérant la route `/product`.  
- `src/Form/Type/ProductPurchaseType.php` — formulaire Symfony (`quantity`, `color`).  
- `templates/product/show.html.twig` — template Bootstrap affichant le produit et le formulaire.  
- `.gitignore` — fichiers exclus du suivi Git.
